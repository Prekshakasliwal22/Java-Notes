* String classes*
-----------------------------------------------------------------------------------------------------------------------------------------
Java provides three type of string classes :
1. String class
2. StringBuffer class
3. StringBuilder class

all above three classes are used to create the object to stored the string (text).

1. String class :
Creates the string object in which we can store the string.
String class contains / holds the string which are immutable.

2. StringBuffer class :
Creates an object in which we can stored the string which is in mutable format.

3. StringBuilder class :
Is used to create the object in which we can stored the string such a string are mutable.

**mutable**
 We can make the changes into the string, we can change it's particular char, we can truncket with any particular character.we can replace it, we can add new char in string etc.

**immutable**
 string are immutable it means that onces string is created we can not make any changes in such a string, we can not change/modify any individual chaaracter of string, we can not delete/truncket any any specific part of the string, we canno insert any new characters in string.


Comparision(difference) between string class, StringBuffer class and StringBuilder class.
-----------------------------------------------------------------------------------------------------------------------------------------
1. After performing any operation on string -- "string class" creates separete copy of the string every time whereas  StringBuffer class  and StringBuilder class creates only one copy, does not creates seperate copy every time.

2. string class can waste the main memory space  because of creating seperate copies of strings. StringBuffer class and StringBuilder class does not waste the main memory.

3. String class provides large number of methods to perform various operations of strings. So that, coding becoms efficient and effective. 
StringBuffer class and StringBuilder class does not provides too many methods.

4.Working with String class becomes easy for understanding, development and efficient to use.
StringBuffer class and StringBuilder class are less efficient.

5. String class can read, contains user inputs. So that, coding becomes generalised and user friendly.
StringBuffer class and StringBuilder class does not contains user inputs.

6. String class is generally use in small program, applications. 
StringBuffer and StringBuilder class are generally used in the development of network application, web appliction, multiuser application.

7. String class does not support from multi-threading so can not be used in any thread programming.
StringBuffer class can be use in multithreaded application in synchronous format which is safe to use and implements.

8. String traversing speed of string class is slower than StringBuffer & StringBuilder class.

9.  string class allows to  create the string variable , to initilize the string variable, to create the string object,to initilize the string object.
string builder class and string buffer class allows only to initilize the string objects

8. equalsIgnoreCase() method
------------------------------------------------------

These two methods are used to check wheather two strings are exactly same (equal) or not?
equals() method check the case of every character during comparistion and  consider the difference between uppercase and lowercase letters. 

equalsIgnoreCase() method does not consider the difference between uppercase & lowercase lettersduring comparing two strings.
It Ignores the case differences.
Syntax:
Stringone.equals(stringtwo)
Stringone.equalsIgnoreCase(stringtwo)

Example:

System.out.println("No, string s1 and s2 are  not same");

System.out.println("Yes, string s1 and s2 are same");
else 
System.out.println("No, string s1 and s2 are  not same");

if(s1.equals(s3))
System.out.println("String s1 and s3 are equal");
else

11. compareToIgnorecase() method
---------------------------------------------------------------

These methods are used to compare two strings and returns zero or positive value or negative value.
When both strings are exacutly same/equal then this method returns 0.
When both strings are not same/equal and first string is greater than second string then this method return positive value.
When both strings are not-exactly same/second string greater than first string this method return negative value.
Positive value or negative value is nothing but --difference between ASCII values of first unmatched characters of both strings.

4.replace() method
------------------------------
This method is used to replace specific part ( from start-index upto end-index) of string by another string.
replace() method can work on only those Strings that are created using Stringbuffer class or StringBuilder class.
replace() method cannot work on Strings of "String class".

syntax:
String1.replace(start-index,end-index,String2)

Example:
// program using replace() method
class StringMethodDemo21
{
 public static void main(String args[])
 {
    StringBuffer sb1=new StringBuffer("Basic Java");
    System.out.println("String sb1: "+sb1);
    sb1.replace(1,4,"python");
    System.out.println(" After replace(),String sb1 is : "+sb1);
    StringBuilder sb2=new StringBuilder("Adavnced Java");
   System.out.println("String sb2: "+sb2);
    sb2.replace(9,13,"c++");
    System.out.println(" After replace(),String sb2 is : "+sb2);
    
}
}

5. reverse() method
------------------------------
This method is used to reverse the string .
reverse() method can work on only those Strings that are created using Stringbuffer class or StringBuilder class.
reverse() method cannot work on Strings of "String class".

syntax:
String.reverse()

Example:
// program using reverse() method
class StringMethodDemo22
{
 public static void main(String args[])
 {
    StringBuffer sb1=new StringBuffer("Java");
    System.out.println("String sb1: "+sb1);
    sb1.reverse();
    System.out.println(" After reverse(),String sb1 is : "+sb1);
    StringBuilder sb2=new StringBuilder("python");
   System.out.println("String sb2: "+sb2);
    sb2.reverse();
    System.out.println(" After reverse(),String sb2 is : "+sb2);
    
}
}
